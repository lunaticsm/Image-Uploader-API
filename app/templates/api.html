AlterBase CDN API

Upload files and retrieve shareable URLs with a minimal HTTP API. Each client is limited to $rate_limit requests per minute and individual uploads are capped at $max_file_mb MB.

Base URL

https://cdn.lunaticsm.web.id

Endpoints

  • POST /upload
    Multipart form request with file field (max $max_file_mb MB). Returns JSON metadata with id, url, size, type.
  • GET /list
    Lists stored files ordered by newest first, including original names and timestamps.
  • GET /{filename}
    Serves the stored file given the UUID filename returned from /upload. Responses include cache headers (Cache-Control: public, max-age) for CDN friendliness.

CURL Example

curl -X POST https://cdn.lunaticsm.web.id/upload \
  -F "file=@/path/to/photo.jpg" \
  -H "Accept: application/json"

Notes

  • Files are stored on disk under the configured uploads directory.
  • The optional cleaner job removes files older than the configured retention window.
  • Use the relative URL from /upload responses to build your CDN links.
